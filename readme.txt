This project is the official implementation code repository for the ICLR 2026 submission paper "Geo-Mamba: A Dual-path Riemannian State Space Model for Functional Organization".
Directory structure description

dataset: It contains the relevant code for data loading and preprocessing.

model: Implement the dual-path Riemann state space model proposed in the paper.

main program (main) : The main running script used for training and evaluating models.

data/ : Stores partition files from the Taowu dataset.

Environment.yml: Used to configure the Conda environment required to run this code.
